SheLu PMOS 新主题素材包

文件说明：
asset_sheet.png              综合资产总表
style_reference.png         风格参考板
home_hero.png               首页主视觉
organ_pack.png              器官元素包（子宫附件/卵巢/骨盆）
decoration_pack.png         装饰元素包（花/星星/轨道/波线/小图案）
app_illustration_pack.png   应用功能插画包（饮食/运动/周期/报告等）

建议上传路径：public/assets/theme/herui/

替换旧皮肤时：
1. 删除旧的 4 套皮肤文件夹或在 theme registry 中取消注册。
2. 新建 herui 主题目录并上传以上素材。
3. 在主题配置中将颜色、插画、装饰统一改为 herui 资源。
4. 不要把 asset_sheet.png 当整页背景使用，它是参考与拆分总表。
